# `AGENT setup`

Initialize `~/.clawdbot/AGENT.json` and the agent workspace.

Related:
- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
AGENT setup
AGENT setup --workspace ~/clawd
```

To run the wizard via setup:

```bash
AGENT setup --wizard
```
